# LinguaMate 翻譯助手（Chrome 擴充功能）

這個專案是一個簡易的 Chrome 擴充功能，使用 OpenAI 的 Chat Completions API（模型固定為 GPT-5-nano）把網頁上的文字翻譯成台灣慣用的繁體中文。點擊擴充功能圖示即可在原本文字區塊下方插入翻譯結果，保持原有的段落結構。

## 功能特色

- **一鍵翻譯**：點擊工具列圖示後，腳本會掃描常見段落元素（標題、段落、清單等）並將翻譯結果插入在原文下方。
- **自訂提示詞**：預設提示詞為「請幫我把下列內容翻譯成台灣慣用的繁體中文」，可在設定頁中調整。
- **本機儲存金鑰**：OpenAI API Key 儲存在瀏覽器的 `chrome.storage.local` 中，不會離開使用者的裝置。

## 安裝步驟

1. 下載或複製此專案原始碼（可使用 `git clone` 或直接下載 ZIP 並解壓縮）。
2. 於 Chrome 的擴充功能管理頁面 (`chrome://extensions/`) 開啟「開發人員模式」。
3. 點擊「載入未封裝項目」，選擇此專案的資料夾，即可完成安裝。

## 使用方式

1. 於 OpenAI 取得有效的 API Key，且該帳戶必須有權存取 `gpt-5-nano` 模型。
2. 進入擴充功能的「選項」頁籤，填入 API Key、確認模型選擇為 GPT-5-nano，並視需要修改提示詞。
3. 開啟任意網頁，點擊擴充功能圖示即可在原文區塊下方看到翻譯結果。

## 開發資訊

- 主要檔案說明：
  - `manifest.json`：Chrome 擴充功能的設定檔。
  - `background.js`：註冊點擊事件並注入翻譯腳本。
  - `contentScript.js`：收集頁面文字並向 OpenAI 送出翻譯請求，再把翻譯結果插入頁面。
  - `options.html`、`options.js`、`options.css`：設定頁面與樣式。
- 請自行妥善保管與使用 OpenAI API Key，避免濫用與額外費用。

## 注意事項

- 翻譯請求會把頁面上偵測到的文字內容送至 OpenAI 伺服器，請在信任的網頁環境使用。
- 若遇到錯誤訊息，可按 `F12` 開啟開發人員工具查看主控台的錯誤詳細資訊。
